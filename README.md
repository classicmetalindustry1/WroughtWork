# The Classic Metal Industry

Welcome to the repository for **The Classic Metal Industry** website.

## Overview
This repository contains the front-end code (HTML, CSS) for a premium, luxury-focused bespoke wrought iron business.

## Tech Stack
- HTML5 (Semantic Structure)
- Vanilla CSS (Flexbox, Grid, Custom Variables)
- Google Fonts (Playfair Display, Montserrat)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser. No build tools are required for the static files.

## Project Structure
- `/index.html`: Main landing page with full copy.
- `/css/styles.css`: Stylesheet containing all design tokens, variables, and responsive layout instructions.

## Design Concept
- **Colors**: Matte Black, Rich Gold, Crisp White, Warm Stone.
- **Vibe**: Elegant, timeless, authoritative, expensive.
